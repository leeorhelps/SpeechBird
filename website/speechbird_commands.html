Just type text"Just Type (text)""Stream Ender" attribute: This command is streamable, but when streamed together with other commands, it can only come as the last command in a stream since any subsequent commands spoken in same breath will be typed out as words rather than executed.

"Selecting" attribute means adding "S" will select the previous word, and Just Type over it.
"Multiplier" attribute means you can overwrite mulitple words e.g: "4 Just TypeS that's nice!" will replace the last 4 words with "that's nice!"
"Ordinal" attribute means you can select a specific last-word to overwrite, e.g: "3rd Just TypeS I like that" will replace the 3rd-last-word with "I like that".

Hints:
"Just Type" is similar to "Type" ", but bypasses WSR's regular dictation engine, meaning:

(A) Even if current app normally doesn't support dictation, you can "Just Type" into it!
(B) If text is selected, it will be replaced with the "Just Type"d text
(C) Will "Just Type", even when speaking a phrase that would normally be a command (e.g: "Just Type 3 Right" will just type "three right", those words, rather than the usual execution, which would be to press the Right Arrow key three times).
(D) "Just Type" handles punctuation and capitalization crudely (unlike "Type", which uses WSR's dictation engine), but "Just Type" works anywhere, including where "Type" and any WSR dictation does not.		Stream Ender, Selecting, Multiplier, Ordinal
